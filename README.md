## Sobre o Projeto

O **Connect+** é um sistema desenvolvido com o objetivo de melhorar a gestão de projetos, tarefas, equipes e produtividade dentro de um ambiente organizacional.  

A proposta do sistema é reunir, em uma única plataforma, recursos que auxiliem no acompanhamento de atividades, na organização das demandas e na visualização do desempenho dos usuários e projetos.

---

## Objetivo

O projeto busca oferecer uma solução que permita:

- gerenciar usuários e equipes;
- cadastrar e acompanhar projetos;
- controlar tarefas e andamento das atividades;
- visualizar indicadores e informações em dashboards;
- centralizar documentação e organização do sistema.
